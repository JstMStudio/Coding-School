Community

Coding School is built for and around a programmer community. Like any other website, Coding School currently has some official communities on multiple platforms. Coding School has communities across Revolt.chat, Guilded and Facebook.

Coding School community is currently moderated and powered by One Future Unified Team and volunteer users of Coding School. Coding School community is a place for everyone using Coding School to gather and hangout and for users to ask for support and give feedbacks.
