About Us

Coding School is a free open source coding website. A website for everyone to learn about coding and programming to built some creative stuff. We currently provides guides for coding languages like HTML (HyperText Markup Language) and CSS (Cascading Style Sheets).

Coding School is a free website for everyone, with no premium membership. Coding School is an open source software, all of the codes for Coding School website is available publicly for everyone to use, study, change, and distribute the codes used to built this website. Coding School website codes is licensed under MIT open source software licenses.
